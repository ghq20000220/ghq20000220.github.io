利用github.io搭建blog

环境（windows版本）

  1. git安装

    • 官方地址：https://git-scm.cn/install/windows
    • 安装后使用cmd或git bash输入git -v，输出版本，说明安装成功

  2. nodejs安装

    • 官方地址：https://nodejs.org/en/download/
    • node -v，显示版本说明成功安装

  3. Hexo安装

    • npm install hexo -g
    • hexo -v，测试安装状态

  4. 在blog仓库文件夹下安装Hexo依赖

    • npm install --save hexo-deployer-git

git ssh配置

  1. 生成ssh key

    • ssh-keygen -t rsa -C "邮件地址" ，连续回车。生成的内容在id_rsa.pub中

  2. 将ssh key加入githu中

    • 使用ssh -T git@github.com测试ssh key的状态

  3. 配置git user.name user.email

初始化blog仓库

  1. 初始化blog目录
    • hexo init
    • 可能会出现git clone失败问题
    • 可能会出现npm install安装卡住问题，可替换npm源
  2. 生成静态网页: hexo g
  3. 预览网页: hexo s
  4. 发布至github: hexo d

编写blog

  1. 新建blog
    • hexo new 'blog-name'

bug合集

  1. hexo 安装后，无法正常显示图片，需要安装插件
